myFlix API Documentation

Welcome to the documentation for the myFlix API. This RESTful API provides access to information about movies, directors, and genres. Users can create accounts, manage their profiles, and curate a list of favorite movies. The API is built with Node.js, Express, and MongoDB, forming the server-side component of the myFlix web application.

About the API

The myFlix API is the server-side backbone of a full-stack JavaScript web application. It is designed to allow client-side applications (built with React) to interact with a movies database. The API handles data management, authentication, and user operations while ensuring security and scalability.

Features [under construction]

  • Retrieve information about movies, directors, and genres.
  • User registration and profile management.
  • Ability for users to create and maintain a list of favorite movies.
  • Secure authentication and authorization for user data.
